# CryptoRate 前端项目说明书

## 🚀 项目目标
CryptoRate 是一个专为加密货币投资者设计的实时行情监控与资产管理平台。本项目追求**极致的科幻极简美学**开发者**: relifel | **版本**: 1.1.3 | **最后更新**: 2026-03-12

---

## 📝 任务反思与项目改进 (2026.03)

### 本阶段完成任务
1. **[New] 个人中心视觉净化**：彻底重构了 `UserCenter.jsx`。
   - **分栏布局**：引入 12 列 Grid 布局（4:8 分栏），解决布局重心偏移问题。
   - **去沉重感**：移除所有深色背景，统一采用纯白卡片+极浅色边框，提升 SaaS 呼吸感。
   - **细节规范**：优化了头像色调、登出按钮位置及列表项 Padding，确保极致简约。
2. **风格巨变**：成功将 UI 从早期的“科幻赛博”风格平滑迁移至“Hyper-Minimalism”极简金融实验室风格。
3. **交互升级**：实现了行情列表的独立卡片流布局及丝滑的 Accordion 展开动效。
4. **链路连通**：完成了前端登录系统与 Spring Boot 后端的深度联通。

### 项目潜在问题与改进方向
- **[待办] 列表一致性**：`Favorites.jsx` 与 `Assets.jsx` 仍使用传统表格，需尽快将其重构为“卡片式行布局”以匹配全站风格。
- **[改进] 实时图表接入**：当前展开面板中的 K 线图为占位组件，计划接入 `Lightweight Charts`。
- **[优化] 响应式深度调节**：针对小屏移动端，大型概览卡片的排版仍有优化空间。
- **[扩展] 状态管理选型**：考虑将业务逻辑迁移至 `Zustand` 或 `TanStack Query`。

## 🚨 紧急修复记录 (最近更新)
针对主页出现的布局缩水和滚动失效问题，已完成以下核心修复：
1. **全局布局解锁**：移除了 index.css 和 App.jsx 中死锁高度的 100vh 和 overflow: hidden，恢复了正常的页面垂直滚动。
2. **全屏居中对齐**：统一了 Home、Assets、Favorites 及其导航栏的容器宽度，强制使用 max-w-[1400px]，确保在不同分辨率下都能大气居中。
3. **内容裁剪修复**：修复了行情列表展开 K 线图时内容被容器裁剪的问题。

## 📊 主要功能说明
- **📈 大盘行情**：实时追踪各主流加密货币的价格、波动及市值，点击行可展开深度趋势图。
- **💰 我的资产**：记录并管理个人持仓，自动计算亏盈与投资回报率 (ROI)。
- **⭐ 我的收藏**：建立个人监控池，支持设置价格预警区间。
- **🤖 AI 智能分析**：沉浸式对话界面，通过 AI 引擎解读市场动态（模拟功能）。

## 🏁 快速开始
1. 确保安装了 Node.js 环境。
2. 在根目录运行 npm install 安装依赖。
3. 运行 npm run dev 启动本地开发服务器。
