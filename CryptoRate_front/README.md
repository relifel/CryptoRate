
# CryptoRate 前端项目说明书

## 🚀 项目目标
CryptoRate 是一个专为加密货币投资者设计的实时行情监控与资产管理平台。本项目追求**极致的科幻极简美学**与**顺畅的用户交互体验**。

## 🚨 紧急修复记录 (最近更新)
针对主页出现的布局缩水和滚动失效问题，已完成以下核心修复：
1. **全局布局解锁**：移除了 index.css 和 App.jsx 中死锁高度的 100vh 和 overflow: hidden，恢复了正常的页面垂直滚动。
2. **全屏居中对齐**：统一了 Home、Assets、Favorites 及其导航栏的容器宽度，强制使用 max-w-[1400px]，确保在不同分辨率下都能大气居中。
3. **内容裁剪修复**：修复了行情列表展开 K 线图时内容被容器裁剪的问题。

## 📊 主要功能说明
- **📈 大盘行情**：实时追踪各主流加密货币的价格、波动及市值，点击行可展开深度趋势图。
- **💰 我的资产**：记录并管理个人持仓，自动计算亏盈与投资回报率 (ROI)。
- **⭐ 我的收藏**：建立个人监控池，支持设置价格预警区间。
- **🤖 AI 智能分析**：沉浸式对话界面，通过 AI 引擎解读市场动态（模拟功能）。

## 🏁 快速开始
1. 确保安装了 Node.js 环境。
2. 在根目录运行 npm install 安装依赖。
3. 运行 npm run dev 启动本地开发服务器。
